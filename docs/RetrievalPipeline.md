# Retrieval Pipeline

## Dense Retrieval

Dense retrieval uses semantic embeddings generated by:

BAAI/bge-small-en-v1.5

The query is embedded and searched against ChromaDB.

---

## Sparse Retrieval

Sparse retrieval uses BM25 keyword search.

This improves exact keyword matching.

---

## Hybrid Retrieval

Results from dense retrieval and BM25 retrieval are merged.

The project uses:

Reciprocal Rank Fusion (RRF)

---

## Reranking

Final ranking is performed using:

cross-encoder/ms-marco-MiniLM-L-6-v2
